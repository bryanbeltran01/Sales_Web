

# Sistema de Administración del Control de Ventas

![Descripción del GIF](wwwroot/Web_sale.gif)

## Descripción

El **Sistema de Administración del Control de Ventas** es una aplicación  desarrollada con la plataforma .NET, diseñada para gestionar el proceso de ventas . Este sistema ofrece una solución para controlar y supervisar todas las actividades relacionadas con las ventas y la gestión de inventarios .

## Desarrollo

El sistema de administración del control de ventas ha sido desarrollado utilizando las siguientes tecnologías y herramientas:

-  **.NET Core 8**

-  **Entity Framework**

-  **Identity y Roles**

-  **Modelo MVC**

-  **SQL Server**

-  **Bootstrap**

-  **JavaScript**

-  **HTML**

## Características

Solicitudes de Ventas.

Contabilidad.

Administración de inventario.

Gestión de Usuarios

Diseño Responsivo

## Instalación

Sigue estos pasos para instalar el proyecto:

1. Clona el repositorio:

```bash

git clone https://github.com/bryanbeltran01/Sales_Web.git

```

2. Abrir el archivo con Visual Studio

```bash

Sales_Web.sln

```

3. Abre SQL Server Management Studio y carga la base de datos proporcionada. Normalmente, la ruta es: `C:\Program Files\Microsoft SQL Server\MSSQL16.MSSQLSERVER\MSSQL\Backup`. Luego, selecciona la opción "Restore Database" y elige el archivo `SalesWeb.bak` para restaurar la base de datos.
```bash
SalesWeb.bak

```
